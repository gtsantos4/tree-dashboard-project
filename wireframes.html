Dashboard Redesign Wireframes

These wireframes show the proposed layout for each page of the Van Yahres dashboard. Click the page tabs above to view each wireframe. Yellow boxes contain design rationale.

📊

KPIs First

Every page leads with a summary KPI row so the client sees the headline numbers before the detail. Current pages bury KPIs or skip them entirely.

📐

Consistent Grid

All pages share the same filter bar, card, and table patterns. Currently each page has its own ad-hoc layout with different spacing and sizing.

🎯

Charts That Earn Their Space

Replace plain data dumps with visual summaries. Add trend sparklines, progress bars, and comparison charts where raw tables don't tell the story well.

🧹

Breathing Room

Current pages are cramped with no visual hierarchy. The redesign uses card containers, consistent padding, section dividers, and a dark totals bar at the bottom of key tables.

Key changes across all pages: (1) Unified filter bar in a white card at top. (2) KPI cards with red accent bars in a grid. (3) Tables inside card containers with sticky headers and a dark totals bar. (4) Charts sized properly and paired with supporting data. (5) Win/Loss badges instead of plain text. (6) Crew Leader breakdown cards instead of raw tables where it makes sense.

Crew Leader Stats

Performance overview by crew leader
